1. What is a Pointer?

A pointer is a variable that stores the memory address of another variable. It

allows for indirect access to the value stored at that address.


2. Pointer Declaration

To declare a pointer, use the * symbol:


    int *ptr;

This means ptr is a pointer to an integer.



3. Dereferencing a Pointer

Dereferencing a pointer means accessing the value at the memory address it points to:


    int value = *ptr;


4. Address-of Operator (&)

The & operator is used to get the address of a variable:


    ptr = &value;

5. NULL Pointer

A NULL pointer is a pointer that doesn't point to any valid memory location. It's

often used to indicate that the pointer is not initialized or that it points to nothing.


6. Void Pointer

A void pointer (void *) is a generic pointer that can point to any data type. It

must be cast to another pointer type before dereferencing.


7. Function Pointer

A function pointer stores the address of a function and can be used to call the function indirectly.


8. Pointer to Pointer

A pointer to a pointer is a variable that stores the address of another pointer:


    int **ptr;

9. Pointer Arithmetic

Pointer arithmetic allows you to perform operations like incrementing or

decrementing pointers, which moves the pointer by the size of the data type it points to.


10. Dynamic Memory Allocation

Functions like malloc(), calloc(), and free() are used for dynamic memory allocation and deallocation.
