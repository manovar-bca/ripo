ðŸ“˜ Introduction to Variables in C


In C, a variable is a named memory location used to store data. Each variable has:


. Data Type: Specifies the type of data it can store.


. Name: The identifier used to reference the variable.


. Value: The actual data stored in the variable.



    
Syntax for declaring a variable:

  
   data_type variable_name = value;
  
example

   int age = 25;

Here, int is the data type, age is the variable name, and 25 is the value.

ðŸ”„ Variable Scope and Lifetime


1. Local Variables


. Scope: Defined within a function or block.


Lifetime: Exists only during the function's execution.



    
example

    void myFunction() {
    int x = 5;
    printf("%d", x);
 }

2. Global Variables


. Scope: Defined outside all functions.


. Lifetime: Exists throughout the program's execution.



    
example

    int globalVar = 10;

  void myFunction() {
    printf("%d", globalVar);
   }
