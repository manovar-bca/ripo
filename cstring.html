1. What is a String in C?


In C, a string is a sequence of characters terminated by a null character


('\0'). It is represented as an array of characters.



        
example

        char str[] = "Hello, World!";

Here, str is a string containing the characters H


, e, l, l, o, ,, a space, W, o, r, l, d, !, and the null character '\0'.



ðŸ”§ Common String Functions in C


C provides several standard library functions to perform operations on strings.


These functions are defined in the header file.


1. strlen()


Calculates the length of a string (excluding the null character).



    
syntex:

        size_t strlen(const char *str);

    


    
example

        
  #include 
  #include 

  int main() {
    char str[] = "Hello";
    printf("Length: %zu\n", strlen(str));
    return 0;
  }
   
output

   Length: 5

2. strcpy()


Copies a string to another.



   char *strcpy(char *dest, const char *src);

    


        
example

        #include 
  #include 

  int main() {
    char src[] = "Hello";
    char dest[20];
    strcpy(dest, src);
    printf("Copied String: %s\n", dest);
    return 0;
  }
   
output

    Copied String: Hello

3. strcat()


Concatenates two strings.



        char *strcat(char *dest, const char *src);

    


        
example

        #include 
   #include 

   int main() {
    char str1[20] = "Hello, ";
    char str2[] = "World!";
    strcat(str1, str2);
    printf("Concatenated String: %s\n", str1);
    return 0;
  }
   
output

   Concatenated String: Hello, World!


4. strcmp()


Compares two strings.



        int strcmp(const char *str1, const char *str2);

    


        
example

        #include 
   #include 

   int main() {
    char str1[] = "Hello";
    char str2[] = "World";
    int result = strcmp(str1, str2);
    if (result == 0)
        printf("Strings are equal.\n");
    else if (result < 0)
        printf("str1 is less than str2.\n");
    else
        printf("str1 is greater than str2.\n");
    return 0;
 }
  
output

   str1 is less than str2.


5. strchr()


Finds the first occurrence of a character in a string.



        char *strchr(const char *str, int c);

    


        
example

        #include 
  #include 

  int main() {
    char str[] = "Hello, World!";
    char *ptr = strchr(str, 'o');
    if (ptr)
        printf("Character found at position: %ld\n", ptr - str);
    else
        printf("Character not found.\n");
    return 0;
  }
  
output

   Character found at position: 4
