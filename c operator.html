In C programming, operators are symbols that perform operations on variables


and values. They are fundamental components of C syntax, enabling developers to


manipulate data and implement logic within programs.


Operators in C are categorized based on the type of operation they perform and


the number of operands they require. Here's an overview of the primary operator categories

1. Arithmetic Operators


These operators perform basic mathematical operations.



| Operator | Description         | Example |
| -------- | ------------------- | ------- |
| `+`      | Addition            | `a + b` |
| `-`      | Subtraction         | `a - b` |
| `*`      | Multiplication      | `a * b` |
| `/`      | Division            | `a / b` |
| `%`      | Modulus (remainder) | `a % b` |
 
    


      
example

      #include 

  int main() {
    int a = 10, b = 5;
    printf("Sum: %d\n", a + b);         // Output: Sum: 15
    printf("Difference: %d\n", a - b);  // Output: Difference: 5
    printf("Product: %d\n", a * b);     // Output: Product: 50
    printf("Quotient: %d\n", a / b);    // Output: Quotient: 2
    printf("Remainder: %d\n", a % b);   // Output: Remainder: 0
    return 0;
  }

2. Relational (Comparison) Operators


These operators compare two values and return 1 if true, 0 if false.



     | Operator | Description              | Example  |
     | -------- | ------------------------ | -------- |
     | `==`     | Equal to                 | `a == b` |
     | `!=`     | Not equal to             | `a != b` |
     | `>`      | Greater than             | `a > b`  |
     | `<`      | Less than                | `a < b`  |
     | `>=`     | Greater than or equal to | `a >= b` |
     | `<=`     | Less than or equal to    | `a <= b` |

   


    
example

    #include 

  int main() {
    int a = 5, b = 3;
    printf("%d == %d: %d\n", a, b, a == b);  // Output: 5 == 3: 0
    printf("%d > %d: %d\n", a, b, a > b);    // Output: 5 > 3: 1
    return 0;
 }

3. Logical Operators


These operators are used to combine multiple conditions.



     | Operator | Description | Example  |            |     |   |     |
     | -------- | ----------- | -------- | ---------- | --- | - | --- |
     | `&&`     | Logical AND | `a && b` |            |     |   |     |
     | \`       |             | \`       | Logical OR | \`a |   | b\` |
     | `!`      | Logical NOT | `!a`     |            |     |   |     |

   


    
example

    #include 

  int main() {
    int a = 1, b = 0;
    printf("a && b: %d\n", a && b);  // Output: a && b: 0
    printf("a || b: %d\n", a || b);  // Output: a || b: 1
    printf("!a: %d\n", !a);          // Output: !a: 0
    return 0;
  }

4. Bitwise Operators


These operators perform bit-level operations.



| Operator | Description | Example    |     |     |
| -------- | ----------- | ---------- | --- | --- |
| `&`      | Bitwise AND | `a & b`    |     |     |
| \`       | \`          | Bitwise OR | \`a | b\` |
| `^`      | Bitwise XOR | `a ^ b`    |     |     |
| `~`      | Bitwise NOT | `~a`       |     |     |
| `<<`     | Left shift  | `a << 2`   |     |     |
| `>>`     | Right shift | `a >> 2`   |     |     |

  


    
example

    #include 

  int main() {
    int a = 5;  // 0101 in binary
    int b = 3;  // 0011 in binary
    printf("a & b: %d\n", a & b);  // Output: a & b: 1
    printf("a | b: %d\n", a | b);  // Output: a | b: 7
    printf("a ^ b: %d\n", a ^ b);  // Output: a ^ b: 6
    printf("~a: %d\n", ~a);        // Output: ~a: -6
    printf("a << 1: %d\n", a << 1); // Output: a << 1: 10
    printf("a >> 1: %d\n", a >> 1); // Output: a >> 1: 2
    return 0;
 }

5. Assignment Operators


These operators assign values to variables.



| Operator | Description         | Example  |
| -------- | ------------------- | -------- |
| `=`      | Simple assignment   | `a = b`  |
| `+=`     | Add and assign      | `a += b` |
| `-=`     | Subtract and assign | `a -= b` |
| `*=`     | Multiply and assign | `a *= b` |
| `/=`     | Divide and assign   | `a /= b` |
| `%=`     | Modulus and assign  | `a %= b` |

  


    
example

    #include 

  int main() {
    int a = 5;
    a += 3;  // a = a + 3
    printf("a += 3: %d\n", a);  // Output: a += 3: 8
    a *= 2;  // a = a * 2
    printf("a *= 2: %d\n", a);  // Output: a *= 2: 16
    return 0;
  }

6. Increment and Decrement Operators


These operators increase or decrease a variable's value by 1.



| Operator | Description | Example        |
| -------- | ----------- | -------------- |
| `++`     | Increment   | `a++` or `++a` |
| `--`     | Decrement   | `a--` or `--a` |




  
example

  #include 

  int main() {
    int a = 5;
    printf("a++: %d\n", a++);  // Output: a++: 5
    printf("++a: %d\n", ++a);  // Output: ++a: 7
    return 0;
  }
